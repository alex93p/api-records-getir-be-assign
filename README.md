## Run the app

### Config the environment

```shell
export NODE_ENV dev
export SERVER_PORT 3000
export SERVER_URL http://127.0.0.1:3000
export MONGODB your_connection_database_string
```

### Install dependencies

```shell
yarn install
```

### Startup server

```shell
yarn run start
```

### Testing

```shell
yarn run test
yarn run test:e2e
```
